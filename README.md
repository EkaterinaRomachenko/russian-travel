Проект: "Путешествиe по России"

В этом проекте, вы окунётесь в потрясающие места России, куда стоит отправиться.

Используются технологии:

1. БЭМ.
2. Flexbox.
3. Grid-layout.
4. Позиционирование блоков и элементов.
5. Figma.
