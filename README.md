Проект: "Путешествиe по России"

В этом проекте, рассказывается об удивительных местах России, которые стоит посетить.

Используются технологии:

1. БЭМ.
2. Flexbox.
3. Grid-layout.
4. Позиционирование блоков и элементов.
5. Figma.
